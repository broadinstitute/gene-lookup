Features:

  • Search genes by name, disease, genomic region, or various gene-disease association attributes
  • Filter by inheritance mode, data source, constraint scores, ClinGen classification, and more
  • Export search results to TSV, BED, or JSON
  • Use IGV.js to explore gene regions alongside reference tracks

GeneLookup aggregates gene-disease association data from the following sources:

  • OMIM
  • ClinGen
  • GenCC
  • PanelApp UK
  • PanelApp Australia
  • DECIPHER
  • ClinVar
  • Fridman et al. 2025
  • gnomAD constraint metrics
It then generates an AI summary of the phenotype descriptions from different sources using Gemini Flash.


All source code is available on GitHub.
